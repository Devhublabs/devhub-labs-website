# DEVHUB LABS WEBSITE

---

# PROJECT OVERVIEW

This project is the official website for DevHub Labs.

The website should represent a modern software engineering team capable of building premium digital products for startups, businesses, organizations and individuals.

This is NOT a template website.

It should feel like the online presence of a world-class software engineering company.

The website should communicate:

- Professionalism
- Technical excellence
- Modern engineering
- Premium quality
- Reliability
- Innovation

Every section should look intentional, polished and memorable.

# HERO SECTION

The Home page Hero is the signature experience of the website.

It should immediately communicate that DevHub Labs is a premium software engineering company.

The Hero should closely follow the artistic direction shown in the reference image.

The Hero contains:

- Sticky Navigation
- Large Headline
- Supporting Paragraph
- Primary CTA
- Secondary CTA
- Trust Indicators
- Animated Hero Object
- Ambient Lighting
- Animated Background
- Scroll Indicator

The Hero should become the strongest visual element of the website.

---

# HERO OBJECT

The DevHub Labs logo is NOT simply an image.

It should feel like a premium product being showcased.

The current logo is located inside:

src/assets/logos/

The implementation must separate:

- Layout
- Animation
- Effects
- Lighting
- Asset

so replacing the logo later requires changing only the image file.

---

# FUTURE 3D LOGO

The future version of the logo will be a professionally rendered 3D asset.

Prepare the Hero so this asset can replace the current logo easily.

The future logo should include:

- Metallic purple material
- Glass reflections
- Beveled edges
- Studio lighting
- Purple glow
- Floating motion
- Mouse interaction
- Soft shadows

Until that asset exists, create the illusion of depth using:

- Perspective transforms
- Multiple shadows
- Layered glows
- Lighting effects
- Floating animations

Never simply rotate the flat PNG like a loading spinner.

# HERO ANIMATIONS

The Hero animation should feel cinematic.

Use:

Slow floating

Very slow rotation

Glow breathing

Mouse parallax

Ambient lighting

Energy pulses

Soft transitions

Respect prefers-reduced-motion.

The motion should feel expensive and elegant.

---

# INTRO ANIMATION

Only on the user's first visit.

Sequence:

Screen begins almost black.

Purple particles appear.

A soft energy pulse expands.

The Hero object materializes.

Glow intensifies.

Headline slides into view.

Buttons fade upward.

Navigation fades in.

Duration:

2–3 seconds.

Do not replay during the same browser session.
